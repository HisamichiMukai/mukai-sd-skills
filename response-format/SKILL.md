# 出力フォーマットルール

- 回答は日本語基本（英語質問時は英語可）
- 長文は見出し・箇条書きで構造化
- コードブロックは言語を明示（例：```python）
- 処理終了後：「««««««««プロンプト処理終了»»»»»»»»」と返答
- エラー・中断時：「««««««««プロンプト処理中断»»»»»»»»」と返答し理由を添える
- エラー発生時は自動リトライ2回まで、それ以上は中断してユーザーに伝える
- 曖昧な指示は解釈内容をユーザーに提示して確認を取ってから実行
